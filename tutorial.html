Welkom bij de tutorial!

In deze tutorial gaan we stap voor stap een spelletje maken. Dit spelletje is een variant op het bekende spelletje "blad, steen, schaar". Het spelletje zal er als volgt uitzien:

Het spelletje zal bestaan uit 2 spelers, de speler en de computer. De speler zal steeds een keuze moeten maken tussen steen, papier of schaar. De computer zal ook een keuze maken tussen steen, papier of schaar. De speler die de beste keuze maakt wint het spelletje.

Om dit spelletje te maken gaan we gebruik maken van HTML, CSS en JavaScript. HTML is de basis van een website, CSS is de stijl van de website en JavaScript is de functionaliteit van de website.

Voor we van start kunnen gaan...

We voegen spelers toe aan de hand van zogenaamde "divs". We geven ze een class mee zodat we ze kunnen stylen en id. De id voor JavaScript!

Wat is een div?

Het DIV element is een voorbeeld van een HTML element. Met dit element kun je een onderdeel van je pagina insluiten en precies daar positioneren waar jij wilt. Dat zie je eigenlijk al aan de naam van het DIV element: 'DIV' is afgeleid van 'division', wat 'verdeling' betekent.

Ga het gevecht aan, doe iets met he blad, de steen of de schaar

Tijd om in actie te schieten. We schrijven nu de blad_steen_schaar functie om ons spel verder tot leven te wekken.

We werken met zogenaamde if, else statements. Dit wilt zeggen: als het ene voorvalt, dan is het andere van toepassing (of vice versa)

Nog enkele verduidelijkingen m.b.t. deze functie:

  • === betekent dat het strikt gelijk is. De keuze van een speler is dus gelijk aan bijvoorbeeld blad.

  • && wilt zeggen dat beide operatoren gelijk moeten zijn. Deze regel code werkt dus enkel als beide factoren tussen de haakjes van toepassing zijn.

  • De -1 wilt zeggen dat er een leventje verdwijnt.
